# 林彥良 (Lin Yen-Liang) - 個人履歷網頁 (純 HTML 版本)

這個專案已轉換為 **純 HTML/CSS/JavaScript** 格式，無需任何建置步驟（No Build Step），可直接使用。

## 如何使用

1. **本地預覽**：
   直接用瀏覽器開啟 `index.html` 即可看到完整效果。

2. **部署至 GitHub Pages**：
   - 建立一個 GitHub 儲存庫（Repository）。
   - 將 `index.html` 上傳至儲存庫根目錄。
   - 前往 **Settings > Pages**。
   - 在 **Build and deployment** 下，將 **Branch** 設定為 `main` (或您的主分支) 並點擊 **Save**。
   - 幾分鐘後，您的網頁將在 `https://<您的使用者名稱>.github.io/<您的倉庫名稱>/` 上線。

## 技術規格
- **Styling**: Tailwind CSS 3 (via CDN)
- **Icons**: Lucide Icons (via CDN)
- **Fonts**: Google Fonts (Inter, Space Grotesk, JetBrains Mono)
- **Performance**: 單檔案載入，原生 JavaScript 處理滾動與動畫。
